# 计算机联锁模拟仿真系统 — 功能说明文档

**技术栈**：Vue 3 + Pinia + Naive UI + Vite  
**运行方式**：`npm run dev` → http://localhost:5173

---

## 一、站场设备配置

本系统模拟一座典型单线中间站，设备配置如下：

### 1.1 轨道区段（7段）

| 编号 | 名称 | 性质 |
|------|------|------|
| JXG | 进站区间 | 区间线 |
| IIAG | 进站咽喉区 | 咽喉 |
| IIG | II号正线股道 | 到发线（正线） |
| 1G | 1号侧线股道 | 到发线（侧线） |
| SFXG | 安全线 | 安全线 |
| IIBG | 出站咽喉区 | 咽喉 |
| JSG | 出站区间 | 区间线 |

每段区段有三种状态：**空闲**（绿色）、**锁闭**（黄色）、**占用**（红色）。

### 1.2 道岔（5组）

| 编号 | 位置 | 定位方向 | 反位方向 |
|------|------|---------|---------|
| 1号 | 进站咽喉 | IIG 正线 | 1G 侧线 |
| 3号 | 进站咽喉（渡线） | IIG 直行 | 1G 渡线 |
| 4号 | 出站咽喉 | IIG→IIBG 汇合 | IIG 渡线 |
| 2号 | 出站咽喉 | 1G→IIBG 汇合 | 1G 延伸 |
| 5号 | 安全线分叉 | 1G 直行 | 安全线 |

每组道岔有四种锁定状态：进路锁（蓝色）、单锁（紫色）、封锁（灰色）、故障（红色）。

### 1.3 信号机（10架）

| 编号 | 类型 | 位置说明 |
|------|------|---------|
| X | 进站信号机 | 进站端主轴上方 |
| D1 | 调车信号机 | 进站端主轴下方 |
| S3 | 出站信号机 | 1G侧线左端上方 |
| SII | 调车信号机 | 1G侧线左端下方 |
| S1 | 出站信号机 | 1G侧线右端下方 |
| XII | 通过信号机 | IIG正线右段上方 |
| X3 | 出站信号机 | IIG正线右端上方 |
| X1 | 出站信号机 | 1G侧线右端上方 |
| D2 | 调车信号机 | 出站端主轴下方 |
| S | 进站信号机 | 出站端主轴上方 |

信号显示：**绿灯**（放行）、**黄灯**（调车）、**红灯**（停止）、**灰色**（断丝故障）。

### 1.4 进路模板（9条）

| 编号 | 名称 | 类型 | 路径 | 涉及道岔 |
|------|------|------|------|---------|
| X-IIG | X→IIG 接车进正线 | 接车 | JXG→IIAG→IIG | 1定、3定 |
| X-1G | X→1G 接车进侧线 | 接车 | JXG→IIAG→1G | 1反 |
| S-IIG | S→IIG 反向接车正线 | 接车 | JSG→IIBG→IIG | 2定、4定 |
| S-1G | S→1G 反向接车侧线 | 接车 | JSG→IIBG→1G | 2反 |
| IIG-X | IIG→X 正线发车上行 | 发车 | IIG→IIAG→JXG | 1定、3定 |
| IIG-S | IIG→S 正线发车下行 | 发车 | IIG→IIBG→JSG | 2定、4定 |
| 1G-X | 1G→X 侧线发车上行 | 发车 | 1G→IIAG→JXG | 1反 |
| 1G-S | 1G→S 侧线发车下行 | 发车 | 1G→IIBG→JSG | 2反 |

## 二、功能模块详述

### 2.1 上位机界面（左侧导航 + 五个面板）

#### 顶部状态栏
- 系统名称与标识
- **实时时钟**（时:分:秒，每秒刷新）
- 系统状态 LED（正常/异常，故障时闪烁）
- 实时统计：活动进路数、在站列车数、故障设备数

#### 站场平面图（SVG 交互图）
- 按真实站场几何绘制：两端单线、中间展开三股道（IIG正线上、1G侧线下、安全线斜下）
- 底层灰色静态轨道线（始终可见）
- 上层彩色状态线（随区段状态变色）
- 道岔圆点随状态变色，圆点下方显示"定位"（绿字）/"反位"（橙字）
- 信号机：粉色方块底座 + 圆形灯头（随信号状态变色）
- 绝缘节标记（蓝色方块，各区段分界处）
- 轨道电路标记（双青色椭圆，两端各一对）
- 坡度标注（6‰）、方向标注（上行/下行）
- **点击任意轨道区段**：直接切换占用/出清状态（仿真快捷操作）
- **点击任意道岔圆点**：弹出道岔操作浮层
- **点击任意信号机方块**：弹出信号机操作浮层
- 在站列车显示为红色矩形图标，标注车次号

---

### 2.2 进路办理面板

#### 进路选排与控制
- 列表展示全部9条进路模板，标注进路类型（接车/发车/调车）、经过区段、道岔要求
- 当前已激活的进路在列表中高亮标注状态
- 操作按钮组（按顺序）：

| 按钮 | 前提条件 | 执行动作 |
|------|---------|---------|
| **锁闭进路** | 进路未激活、路径空闲、无敌对进路 | 自动转岔 → 区段锁闭 → 道岔锁闭 → 添加活动进路 |
| **开放信号** | 进路已锁闭 | 对应信号机显示绿灯（接车/发车）或黄灯（调车） |
| **正常解锁** | 所有区段已出清（无列车占用） | 解除区段锁、道岔锁，信号复红，删除活动进路 |
| **取消进路** | 进路已激活 | 启动30秒倒计时，进度条动画；倒计时结束后自动解锁 |
| **立即解锁（人工）** | 倒计时进行中 | 绕过出清检查强制解锁（人工解锁） |

#### 活动进路信息卡
- 显示进路名称、锁闭状态（锁闭/开放/占用）
- 显示经过区段列表、建立时间、信号机编号、道岔定反位要求

#### 联锁条件检查（自动执行，以下任一不满足则拒绝）
1. **敌对进路检查**：新进路路径与已激活进路路径有交叉，或同一道岔要求位置相反
2. **区段空闲检查**：进路路径中所有区段均未占用、未锁闭
3. **道岔可用检查**：所需道岔未被进路锁、未单锁、未封锁、未故障
4. **出清检查**（解锁时）：进路路径中所有区段均无列车占用

#### 列车作业办理
- 输入车次号，选择作业类型（**接车** / **发车** / **通过**），记录日志

#### 调车作业办理
- 输入调车号、选择起始区段和目标区段，记录调车作业日志

---

### 2.3 道岔控制面板

每组道岔显示一张控制卡，包含：

- 当前位置大字显示（定位/反位，带颜色圆点指示）
- 锁定状态标签（进路锁 / 单锁 / 封锁 / 故障）
- **单独操作**：定位 / 反位按钮（需满足：无进路锁、无单锁、无封锁、无故障）
- **单锁 / 解单锁**：手动锁定道岔当前位置，防止误操作
- **封锁 / 解封锁**：将道岔置为不可用状态
- **故障仿真**：设置故障状态（自动封锁，所有操作拒绝）；清除故障后自动解封

---

### 2.4 信号机面板

每架信号机显示一张控制卡，包含：

- 信号灯圆形显示（绿/黄/红/灰，对应开关/故障）
- 信号机名称、类型标签（进站/出站/调车）
- 故障时显示闪烁红色警告横幅"断丝故障 — 强制显示红灯"
- **人工控制**：绿灯 / 黄灯 / 红灯按钮（故障时禁用；调车型号方可设黄灯）
- **断丝故障仿真**：设置断丝故障（信号强制红灯）；恢复后可重新操作

---

### 2.5 列车管理面板

#### 添加列车
- 输入车次号（最长8位）
- 选择初始所在区段（占用区段灰显不可选）
- 选择运行方向（上行 / 下行）
- 添加后：区段自动标记为占用，站场图显示列车图标

#### 在站列车列表
- 显示每辆列车的车次、方向、当前区段、速度（km/h）
- **移动列车**：点击目标区段按钮，将列车从当前区段移至目标区段
  - 移动前检查是否有进路保护，无保护时发出黄色警告
- **出清列车**：点击 ✕ 按钮，列车删除，所在区段恢复空闲

#### 自动运行
- 启动/停止切换按钮，运行间隔可在 1～30 秒内调节
- 启动后每隔设定秒数，自动将每辆列车随机移至一个空闲区段

#### 列车作业记录表
- 记录每次入站、运行、出站事件
- 显示车次、作业类型、位置/路径、时间戳
- 最多保留100条，超出后自动滚动丢弃旧记录

---

### 2.6 操作日志面板

- 所有操作自动记录，包含时间、操作类型、详情
- 日志分四个级别：成功（绿色左边框）、警告（橙色）、错误（红色）、信息（灰色）
- 支持按级别筛选（全部/成功/错误/警告/信息）
- 显示日志总条数
- **导出**：将当前筛选结果导出为 `.txt` 文件
- **清空**：清除所有日志记录

#### 设备状态概览卡
- 轨道区段：总数 / 空闲 / 锁闭 / 占用
- 道岔：总数 / 正常 / 故障封锁
- 活动进路数、在站列车数

#### 系统全体复位
- 清除所有进路、列车，恢复所有设备到初始状态
- 操作前弹出确认对话框

---

### 2.7 设备点击浮层（站场图直接操作）

在站场图上点击设备，在点击位置弹出快捷操作浮层，无需切换到对应面板：

**道岔浮层**：
- 显示当前位置（定位/反位）和锁定状态标签
- 单独操作（定位/反位）、单锁/解单锁、封锁/解封锁、故障仿真

**信号机浮层**：
- 显示当前灯色（大圆灯）和信号类型
- 人工设置绿灯/黄灯/红灯、断丝故障仿真

**轨道区段浮层**：
- 显示当前状态（空闲/锁闭/占用/封锁）

点击浮层外任意空白处关闭浮层。

## 三、仿真功能

### 3.1 轨道电路仿真
- **列车占压**：添加列车或移动列车至某区段，该区段自动变红（occupied=true）
- **出清仿真**：移出列车或点击区段（快捷切换），区段恢复绿色（occupied=false）
- **直接点击**：在站场图上点击任意区段，立即切换占用/出清状态（用于快速仿真演示）

### 3.2 信号机仿真
- **开放**：进路开放后，对应信号机自动显示绿灯（接发车进路）或黄灯（调车进路）
- **关闭**：进路解锁或取消后，信号机自动复示红灯
- **断丝故障**：设置断丝状态，信号机强制显示红灯并显示故障警告，进路办理时拒绝开放该信号

### 3.3 道岔仿真
- **动作仿真**：单独操作道岔时，圆点颜色及"定位/反位"文字实时切换
- **位置状态**：定位显示绿色，反位显示橙色，锁闭显示蓝色，故障显示红色
- **故障状态**：设置故障后道岔自动封锁，相关进路无法办理，恢复后自动解封

---

## 四、数据持久化

- 所有设备状态、活动进路、在站列车、操作日志（最近100条）自动保存至 `localStorage`
- 刷新页面后自动恢复上次状态
- 系统全体复位时清除所有持久化数据
